# New deias

- A New Ideias é uma plataforma voltada à troca de ideias e negociações de projetos entre usuários. A plataforma em si funciona como um fórum onde qualquer pessoa que estiver disposta a compartilhar suas ideias pode posta-las na plataforma. A mesma conta com abas de comunidade, que permitem maior interação entre os usuários, além de possibilitar a análise de como cada ideia repercute na comunidade, facilitando o desenvolvimento de projetos open-source e iniciativa de formação de equipes de desenvolvimento para diversas finalidades.
